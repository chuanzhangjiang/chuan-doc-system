## ubuntu服务器相关常用命令
1、使用ssh登录远程服务器
```
ssh {登陆服务器所用的用户名}@{服务器IP地址}
```
## docker 常用命令
1、运行容器
```
sudo docker container start {容器名字}
```
2.列出所有容器
```
sudo docker container ls -al
```
3.进入正在运行容器的bash界面
```
sudo docker container exec -it {容器名} bash
```
## mysql常用命令
1.进入mysql命令终端
```
mysql -u {用户名} -p
```
